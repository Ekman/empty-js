# empty php js - empty() like in php, but in js

## Just install empty-php-js from NPM:

```
npm i empty-php-js
```

## Require it in your JavaScript project with name "empty":

```
const empty = require('empty-php-js');
```

## And use it:

```js
if(empty('') || empty({})){
//...
}
```
